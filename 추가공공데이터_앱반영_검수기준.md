## 2. 기본 유지 기준

장소를 유지하려면 아래 중 하나 이상을 만족해야 한다.

- 원자료 소개, 프로그램, 시설명, 축제내용에 교육과정 키워드가 직접 포함된다.
- 장소 유형이 단원 활동과 직접 연결된다.
- 역사 단원에서는 시대·사건·인물·유물·유적 근거가 있다.
- 자연·지리 단원에서는 산지, 하천, 해안, 생태, 지형, 탐방로, 지도 활용 근거가 있다.
- 촌락·경제 단원에서는 농어촌체험, 생산, 소비, 지역 특산, 마을 공동체 근거가 있다.
- 가족 방문 활동으로 전환 가능한 체험·탐방·관찰 요소가 있다.

## 3. 기본 제외 기준

다음은 앱 추천에서 제외한다.

- 음식점, 카페, 숙박, 펜션, 호텔, 모텔, 쇼핑, 상가, 일반 매장
- 단순 어린이공원·소공원 중 교육 근거가 없는 곳
- 주소·좌표가 모두 없고 장소 식별이 어려운 후보
- 역사 단원에 시대 태그가 맞지 않는 장소
- 5학년 6단원에 독립운동, 의병, 항일, 3·1운동, 민주주의, 6·25, 국가수호 근거가 없는 장소
- 5학년 6단원 기본 장소 추천에는 일반 지역축제·먹거리축제·전년도 참고 축제를 넣지 않는다.
  도시공원 보유시설에 `충혼탑`처럼 기념 시설명이 단독으로 들어간 경우는 5학년 6단원 직접 근거로 보지 않는다.
  `6·25`, `전쟁`, `호국`, `국가수호`, `전몰`, `순국열사`, `현충시설` 등 사건·인물·시대 근거가 함께 확인될 때만 추천 후보로 인정한다.
  `평화시장`처럼 장소명에 들어간 일반 고유명사의 `평화`는 호국·전쟁 근거로 보지 않는다.
- `주왕산`처럼 지명 안에 우연히 포함된 `왕산`은 독립운동가 왕산 허위 관련 근거로 보지 않는다.
  `왕산 허위`, `왕산허위`, `허위`처럼 인물 맥락이 확인될 때만 5학년 6단원 근거로 인정한다.
- 독립운동 인물 시설은 장소명 또는 원자료 필드에 인물명이 직접 확인될 때만 근거로 쓴다.
  예: `남자현지사생가`, `남자현지사기념관`, `추모각`, `추모비`가 확인되면 5학년 6단원 직접 근거로 인정한다.
- 축제가 이미 종료되었는데 현재 진행 행사처럼 보이는 후보

| sourceType | 검수 포인트 |
|---|---|
| ecoTour | 좌표 보강 필요. 자연·생태 단원에만 우선 적용 |
| cityPark | 단순 생활공원 오염 주의. 생태·수변·역사·문화·공공시설 근거가 있어야 유지 |
| nationalParkTrail | 코스명, 난이도, 통제 여부 확인. 지도에는 대표점만 먼저 표시 |
| forestTrail | 주소 없음. 산명 기반 후보라 좌표 변환 또는 별도 지도 검수 필요 |
| ruralVillage | 체험 프로그램 중심으로 유지. 먹거리체험은 음식점 추천으로 표현 금지 |
| festival | 날짜 상태 필수. 전년도 자료는 참고자료로만 표시. 근현대사 단원 기본 장소 추천에서는 제외하고, 필요 시 별도 `역사 행사 참고` 영역으로 분리 |
| touristArea | 일반 관광지 오염 주의. 소개 필드에 교육 키워드가 있어야 유지 |
| museumArt | 전시 주제와 단원 연결 확인. 단순 미술 감상은 보조자료로 낮춤 |

## 5. 앱 표시 문구 기준

장소명 보정:

- 공공데이터 원자료명과 실제 지도 서비스·현장 표기가 다를 수 있다.
- 화면 표시명은 사용자가 찾기 쉬운 명칭으로 보정하되, 팝업에는 원자료 장소명을 함께 남긴다.
- 예: `왕산공원` → 화면 표시 `왕산역사공원`, 원자료 장소명 `왕산공원`

직접 연계:

- 원자료 필드와 단원 키워드가 실제로 겹칠 때만 사용한다.
- 예: `체험프로그램=갯벌체험, 어업체험` → 촌락·생산·소비 단원 직접 연계

보조 연계:

- 장소는 유용하지만 단원 핵심 개념과 직접 일치하지 않을 때 사용한다.
- 예: 일반 관광지, 종료된 축제, 단순 문화 체험

검수 필요:

- 좌표·주소·시기·장소 유형 검증이 더 필요한 경우 사용한다.
- 앱에서는 추천 결과 하단 또는 토글로 분리한다.

AI 보조 문장:

- 원자료 필드에 확인되는 장소명, 시설명, 분류, 소개만 바탕으로 작성한다.
- 단원 키워드를 장소 근거처럼 새로 만들어 넣지 않는다.
- 직접 근거가 부족하면 추천 이유를 생성하지 않고 `직접 연계 근거 부족`으로 표시한다.
