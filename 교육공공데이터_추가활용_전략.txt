교육 공공데이터 추가 활용 전략
작성일: 2026-05-11

============================================================
현재 진단
============================================================

현재 앱에서 실제 기능에 연결된 교육 공공데이터는 사실상 아래 1개가 핵심이다.

1. NEIS 학교기본정보 Open API
   - 학교 검색
   - 학교 주소 확인
   - 학교 기준 위치/거리 계산
   - 학교 시군 기반 지역화 추천

이 데이터는 필수 요건 충족에는 의미가 있지만,
대회 심사에서 "교육 공공데이터 활용성"을 강하게 어필하기에는 다소 약하다.

따라서 교육 공공데이터를 단순히 1개 붙이는 수준이 아니라,
학교의 실제 운영 맥락, 학사 일정, 시간표, 학교 여건을 추천 과정에 반영하는 구조로 확장하는 것이 좋다.


============================================================
우선 추가해야 할 교육 공공데이터
============================================================

[1] NEIS 학사일정 Open API

- 공식 데이터명
  교육부_나이스(NEIS) 교육정보 개방 포털_학사일정

- 제공기관
  교육부

- 데이터 성격
  교육 공공데이터

- 주요 제공 항목
  시도교육청명
  행정표준코드
  학교명
  학년도
  학교과정명
  수업공제일명
  학사일자
  행사명
  행사내용
  학년별 행사여부

- 앱 적용 아이디어
  1. 체험학습 추천 시 학교의 실제 학사일정을 확인한다.
  2. 현장체험학습, 수학여행, 재량휴업일, 시험기간, 방학 등과 겹치는지 판단한다.
  3. 추천 결과에 "학사일정상 운영 가능 시기" 또는 "체험학습 후보 주간"을 표시한다.
  4. 발표에서는 "단순 장소 추천이 아니라 학교 일정 데이터와 결합한 실행 가능한 추천"이라고 설명한다.

- 앱 기능 문구 예시
  선택한 학교의 NEIS 학사일정을 분석하여 체험학습 운영 가능성이 높은 시기를 함께 제안합니다.

- 추천 우선순위
  매우 높음


[2] NEIS 초등학교 시간표 Open API

- 공식 데이터명
  교육부_나이스(NEIS) 교육정보 개방 포털_초등학교 시간표

- 제공기관
  교육부

- 데이터 성격
  교육 공공데이터

- 주요 제공 항목
  학년도
  학교
  학기
  학년
  학급
  교시별 시간표
  수업내용

- 앱 적용 아이디어
  1. 사용자가 학년을 선택하면 해당 학년 시간표 데이터를 조회한다.
  2. 사회 수업이 있는 요일이나 교시를 확인한다.
  3. 해당 단원의 수업 운영 흐름과 체험학습 추천을 연결한다.
  4. "단원-장소 추천"에서 한 단계 더 나아가 "실제 학교 시간표 기반 수업 적용 가능성"을 보여준다.

- 앱 기능 문구 예시
  선택한 학교의 시간표 데이터를 참고하여 사회 수업과 연계하기 쉬운 체험학습 운영 방식을 제안합니다.

- 추천 우선순위
  매우 높음

- 주의점
  학급 정보가 필요할 수 있으므로 UI에 반 선택 항목이 추가될 수 있다.
  대회용 데모에서는 "선택 학교/학년/학급 기준 시간표 연동 가능"을 보여주는 것만으로도 의미가 크다.


[3] NEIS 급식식단정보 Open API

- 공식 데이터명
  교육부_나이스(NEIS) 교육정보 개방 포털_급식식단정보

- 제공기관
  교육부

- 데이터 성격
  교육 공공데이터

- 주요 제공 항목
  급식일자
  식사명
  메뉴명
  원산지 정보
  칼로리 정보
  영양 정보
  알레르기 정보

- 앱 적용 아이디어
  1. 체험학습 후보일의 급식 여부를 확인한다.
  2. 급식이 없는 날, 현장체험학습일, 외부 활동 가능일을 추정하는 보조 데이터로 활용한다.
  3. 장거리 체험학습 추천 시 점심 운영 방식, 도시락 필요 여부를 표시한다.
  4. 학생 안전/건강 요소와 연결해 알레르기 정보 확인 기능으로 확장할 수 있다.

- 앱 기능 문구 예시
  후보 체험학습일의 급식 운영 여부를 확인하여 도시락 준비나 외부 식사 필요 여부를 안내합니다.

- 추천 우선순위
  중간

- 주의점
  앱의 핵심은 교육과정-관광자원 매칭이므로 급식은 보조 기능으로만 쓰는 것이 좋다.


[4] 학교알리미 공시정보 Open API

- 공식 데이터명
  한국교육학술정보원_학교알리미 공시정보

- 제공기관
  한국교육학술정보원

- 데이터 성격
  교육 공공데이터

- 주요 제공/공시 항목
  학교기본정보
  수업일수 및 수업시수 현황
  학교 현황
  성별 학생수
  학년별·학급별 학생수
  교원 현황
  학생교육활동 지원시설 현황
  시설안전 점검 현황
  급식 실시 현황
  안전교육 계획 및 실시현황
  장학금 수혜 현황 등

- 앱 적용 아이디어
  1. 학교 규모, 학급 수, 학생 수를 참고해 체험학습 운영 난이도를 추정한다.
  2. 학년별 학생 수가 많으면 대형 수용 가능한 장소를 우선 추천한다.
  3. 안전교육 계획 또는 시설안전 관련 공시 항목과 연결해 안전 체크리스트를 강화한다.
  4. 수업시수/수업일수 데이터를 참고해 체험학습 계획의 현실성을 설명한다.

- 앱 기능 문구 예시
  학교알리미 공시정보를 참고하여 학교 규모와 교육여건에 맞는 체험학습 운영 방식을 제안합니다.

- 추천 우선순위
  높음

- 주의점
  학교알리미 API는 별도 인증키/이용 절차가 필요할 수 있으므로,
  대회 제출 전 연결 가능 여부를 확인해야 한다.


============================================================
앱에 넣었을 때 가장 설득력 있는 조합
============================================================

추천 조합 A: 빠르게 강해지는 조합

1. NEIS 학교기본정보
2. NEIS 학사일정
3. NEIS 초등학교 시간표
4. 한국관광공사 Tour API
5. 교과서 PDF 분석 데이터

어필 문장:
본 서비스는 학교기본정보, 학사일정, 초등학교 시간표 등 교육 공공데이터를 결합하여
선택한 학교의 실제 교육 운영 맥락 속에서 경북 관광자원을 교육과정 단원과 연결한다.


추천 조합 B: 심사위원에게 데이터 활용성이 크게 보이는 조합

1. NEIS 학교기본정보
2. NEIS 학사일정
3. NEIS 초등학교 시간표
4. NEIS 급식식단정보
5. 학교알리미 공시정보
6. 한국관광공사 Tour API
7. 교과서 PDF 분석 데이터

어필 문장:
본 서비스는 학교 위치, 학사일정, 시간표, 급식 운영, 학교 규모와 같은 교육 공공데이터를 종합해
체험학습의 교육과정 적합성뿐 아니라 실제 운영 가능성까지 판단하는 AI 기반 추천 시스템이다.


============================================================
기능 반영 우선순위
============================================================

1순위: NEIS 학사일정
  - 이유: 체험학습 운영 가능 시기와 직접 연결된다.
  - 구현 난이도: 중간
  - 발표 효과: 높음

2순위: NEIS 초등학교 시간표
  - 이유: 사회 수업과 실제 수업 운영 흐름을 연결할 수 있다.
  - 구현 난이도: 중간~높음
  - 발표 효과: 매우 높음

3순위: 학교알리미 공시정보
  - 이유: 학교 규모와 여건을 반영할 수 있어 데이터 활용 폭이 커 보인다.
  - 구현 난이도: 중간~높음
  - 발표 효과: 높음

4순위: NEIS 급식식단정보
  - 이유: 체험학습 당일 운영 보조 정보로 활용할 수 있다.
  - 구현 난이도: 낮음~중간
  - 발표 효과: 중간


============================================================
추천되는 앱 화면 변화
============================================================

1. 학교 선택 후 "학교 데이터 분석" 패널 추가
   - 학교 기본정보
   - 학사일정 조회 여부
   - 시간표 조회 여부
   - 급식 조회 여부

2. 추천 결과 팝업에 "운영 가능성" 영역 추가
   - 학사일정 기반 후보 시기
   - 사회 수업 연계 가능 요일
   - 급식/도시락 고려 사항
   - 학교 규모에 따른 운영 팁

3. 발표용 데이터 처리 과정 추가
   - 교육 공공데이터 수집
   - 학교 맥락 데이터 전처리
   - 교육과정 단원 전처리
   - 관광자원 전처리
   - 교육과정 적합성 + 운영 가능성 통합 추천


============================================================
최종 발표용 핵심 문장
============================================================

기존 버전은 학교기본정보를 중심으로 학교 위치 기반 추천을 제공했지만,
확장 버전은 NEIS 학사일정, 초등학교 시간표, 급식식단정보, 학교알리미 공시정보를 함께 활용한다.
이를 통해 단순히 "가까운 체험학습 장소"를 추천하는 것이 아니라,
선택한 학교의 실제 학사 운영, 수업 시간표, 급식 여부, 학교 규모를 반영하여
교육과정과 지역 관광자원의 연결 가능성과 실행 가능성을 함께 판단한다.
