## 4. 결론

현재 실제 API 연결로 주장해도 되는 것은 `NEIS 학교기본정보`, `한국관광공사 Tour API`, `국가유산청 지정문화재 현황/상세`, `국가보훈부 현충시설정보`, `OpenStreetMap Nominatim`이다.

추가 8종 공공데이터는 원자료를 실제 수집·전처리한 뒤 앱용 정적 데이터로 연결한 상태이다. 보고서에는 "실시간 API 호출"이 아니라 "공공데이터 수집·전처리 후 앱 연결"이라고 표현하는 것이 정확하다.

국사편찬위원회 한국사DB와 2022 개정 사회과 교육과정은 교육적 근거 설계에는 들어가 있으나, 현재 HTML에서 실시간 API로 호출되지는 않는다. 보고서에는 이 둘을 "직접 API 연결"이 아니라 "교육과정/역사 원자료 기반 전처리 기준"으로 표현하는 것이 안전하다.
